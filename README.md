# Eco Crafting Project

Economy crafting game prototype with a React/Vite frontend, Node.js/Vite backend, and Drizzle ORM database layer.

## Apps

- `apps/frontend`: React + TypeScript + Vite
- `apps/backend`: Node.js + TypeScript + Vite + Express + Drizzle
- `packages/shared`: shared game types

## Quick Start

```bash
npm install
npm run check
npm run dev:backend
npm run dev:frontend
```

Create `apps/backend/.env` from `apps/backend/.env.example` when connecting a real PostgreSQL database.
